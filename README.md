# GetImmiHub

Next.js 14+ project with **App Router**, **TypeScript**, and **Tailwind CSS**.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Start (production)

```bash
npm start
```

## Lint

```bash
npm run lint
```

## Project structure

- `src/app/` – App Router routes and layouts
- `src/app/layout.tsx` – Root layout
- `src/app/page.tsx` – Home page
- `src/app/globals.css` – Global styles and Tailwind
- `@/*` – Import alias for `./src/*` (see `tsconfig.json`)

## Tech stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **ESLint** (Next.js config)
